Caesar Cipher

The Caesar Cipher is a simple substitution cipher that shifts each letter of the plaintext by a fixed number of positions down the alphabet. For example, a shift of 3 turns A into D, B into E, and so on.

Additive Cipher

The Additive Cipher encrypts a message by adding a constant value (the key) to each letter’s position in the alphabet. Decryption reverses this process by subtracting the key.

Multiplicative Cipher

The Multiplicative Cipher multiplies each letter’s position in the alphabet by a fixed key. Decryption requires finding the modular multiplicative inverse of the key.

Affine Cipher

The Affine Cipher combines two operations: multiplication and addition. A letter’s position is first multiplied by a key, then a constant value is added. Decryption involves the inverse of these operations.
